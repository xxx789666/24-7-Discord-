新專案的啟動流程如下：

1.環境初始化：/init 生成 claude.md。
2.掛載現成 MCP：請將寫好的通用的 MCP 工具 直接在 .mcp.json 中引入。
3.載入commands
4.載入hooks
5.git 初始化專案 (分散式版本控制系統)後台運行的系統
Install git comand if you haven't. Then, run init for the current projet. Then, set up git user.name as "ax958888" and user.email "astrax958@gmail.com" if those git config are not set yet.


6/writing-plans並在需要執行test-driven-development task 的地方 註記讓agent 執行



7\.再次確認開發準備環境所有工具與套件完整


8./dispatch "閱讀 CLAUDE.md 和專案任務清單，依任務複雜度建立 alias 清單"
9.將 output.md 中的 YAML 複製貼入 \~/.dispatch/config.yaml


10.完成後 在source control 按+ 放入 Staged Changes 設為 first commit。
